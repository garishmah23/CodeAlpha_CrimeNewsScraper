# Crime News Web Scraper

## Project Overview

This project was developed for CodeAlpha Data Analytics Internship Task 1 (Web Scraping).

The project collects crime-related news headlines from a public news website using Python web scraping techniques. The scraped data is stored in a CSV file and analyzed through Python scripts and visualizations.

## Features

* Web scraping using BeautifulSoup
* Data extraction from news websites
* CSV dataset generation
* Basic data analysis
* Data visualization using Matplotlib

## Technologies Used

* Python
* BeautifulSoup
* Requests
* Pandas
* Matplotlib

## Project Structure

Crime-News-Web-Scraper/

├── scraper.py

├── analysis.py

├── visualize.py

├── crime_news.csv

├── requirements.txt

└── README.md

## Installation

Install dependencies:

pip install -r requirements.txt

## Usage

Run scraper:

python scraper.py

Run analysis:

python analysis.py

Run visualization:

python visualize.py
